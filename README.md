# Warriors Plugin Core

### Features

1. Custom Classes.
2. Custom Abilities.
3. Soup Consumption w/ Regeneration effect.
4. Knockback Handler.
5. Energy/Mana.
6. Customized Events to fit all dependency needs.
7. Plugin API, with access to creation of new classes, abilities, and much more.
